CSV layer graph

Upload a nodes CSV and an edges CSV to render a multi-layer semantic zoom graph. Any number of layers is supported.
Upload both CSVs, or click "Load sample data" to see it in action.
How layers work: the layer column is an integer depth (1, 2, 3...). Rows connecting nodes on the same layer are drawn horizontally (dashed); rows connecting different layers are drawn vertically — this is detected automatically, you don't need a "kind" column.
Click a node to drill into the next layer down (only the layer you're on stays fully solid; the layers before it fade into an ancestor trail; everything else stays hidden). Click the breadcrumb or "Reset view" to zoom back out.
